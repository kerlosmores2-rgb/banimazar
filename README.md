# نظام إدارة محطات الصرف الصحي

نظام متكامل لإدارة وتشغيل وصيانة محطات الصرف الصحي بفرع بني مزار - شركة مياه الشرب والصرف الصحي بالمنيا.

## المميزات

- إدارة المحطات (إضافة، تعديل، حذف، عرض)
- إدارة الأصول (مضخات، مولدات، لوحات كهربائية، معدات ميكانيكية، مباني)
- بلاغات الصيانة والأعطال
- تقارير المرور الميداني
- التقارير التشغيلية (كهرباء، سولار، مياه)
- غرامات معامل القدرة وإخطاراتها
- إدارة الموظفين والمستخدمين
- صلاحيات متعددة (Admin, Station Manager, Engineer, Viewer)
- واجهة متجاوبة تعمل على جميع الأجهزة

## التقنيات المستخدمة

- **Backend:** Node.js, Express.js, SQLite3
- **Frontend:** HTML5, CSS3, JavaScript
- **المصادقة:** JWT, bcrypt
